# WakaTime for Termux

Wakatime for termux makes it easier for us to see the working time as a programmer. Time for coding is better so that your time can be seen by the boss, hahaha

---

## Install
Download Termux dif-droid directly from GitHub. Click the link below;

  [F-Droid](https://f-droid.org/packages/com.termux)
  [Github](https://github.com/termux/termux-app/releases)

Download what we provide because the code is always open to everyone 🤝

  Register a wakatime account to start working.
      [login/sign up](https://wakatime.com)
run this bash code then it will be directed automatically

```bash
curl -fsSL https://github.com/nt-portal/Waka-Termux/raw/main/install.sh | bash
```
